# Customer Revenue Risk Dashboard (Power BI)

## Project Overview

Built an interactive Power BI dashboard to analyze sales performance, customer concentration risk, product profitability and salesperson performance.

## Features

* Revenue and profit KPI cards
* Pareto analysis (80/20 rule)
* Customer concentration risk analysis
* Year-over-Year revenue analysis
* Product profitability analysis
* Salesperson performance analysis
* Interactive slicers and filters

## Tools Used

* Power BI
* DAX
* CSV datasets
* Data modeling

## Key Insights

* Identified revenue dependency on top customers.
* Analyzed product profitability trends.
* Evaluated salesperson performance.
* Monitored business growth over time.
